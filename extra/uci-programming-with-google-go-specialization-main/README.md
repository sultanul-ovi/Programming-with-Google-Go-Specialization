# UCI Programming with Google Go  

## 课程简介  

- 所属大学：UC Irvine  
- 先修要求：无  
- 编程语言：Go  
- 课程难度：🌟🌟  
- 预计学时：50小时  

UC Irvine在Coursera上出的Go语言入门课程，分成了三门课程来讲授Go语言的基础语法，特性和并发。  
一门课程分四周，每周都会有测验，作业和学生互评。文档齐全，适合自学。
